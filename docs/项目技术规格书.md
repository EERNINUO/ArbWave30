## 1. 概述
本设备是一款基于高云FPGA和高速DAC的数字合成信号源，为解决作者在学习实践中缺少高速信号源的问题而启动，旨在提供高精度、宽带宽的可编程波形输出。支持USB远程控制，并可扩展外部参考时钟输入，适用于实验室测试、教学演示及嵌入式系统调试等场景。

## 4. 功能要求

### 4.1 基础功能
- 产生正弦、方波、三角波、锯齿波、阶梯波、直流
- 可设置频率、幅值、偏移、相位
- 支持输出开关（ON/OFF），断电后恢复最后设置（存储于EEPROM）

### 4.2 调制与扫频（V2版本实现）
- AM：内/外调制，调制频率/深度可设
- 线性/对数频率扫描

### 4.3 任意波功能（V2版本实现）
- 用户自定义波形点数：最大 16kpts（14bit分辨率）
- 通过上位机软件或SCPI命令下载波形数据

### 4.4 外部接口
- USB 2.0（USBTMC或CDC类）用于远程控制和波形数据传输
- 波形输出：SMA接口，50Ω/高阻切换
- 外部参考时钟输入（10 MHz，SMA接口）
- 触发输入/输出（SMA）
- 电源输入（DC2.1/2.5mm 24~36 V）
- 可选：LAN接口（后续实现）

### 4.5 远程控制
- 接口：USB 2.0 (USBTMC或CDC类)
- 命令集：符合SCPI-1999标准，至少包含以下子系统：
  - `*IDN?`, `*RST`, `*CLS`
  - `SOURce`：FREQuency, VOLTage, VOLTage:OFFSet, FUNCtion, PHASe
  - `OUTPut`：STATe, IMPedance, LOAD
  - `TRIGger`：SOURce, SLOPe
  - `SYSTem`：ERRor?
  - `SOURce`:TRACe:DATA?
- 响应格式：`<header><value><terminator>`，支持查询与设置

### 4.6 用户界面（长期扩展）
- 可选：通过USB连接PC运行简易GUI（Python+PyQt）
- 可选：后续移植至Linux板卡（运行本地GUI）

## 5. 硬件架构要求

### 5.1 主要IC选型（初步）
| 模块 | 型号 | 关键参数 | 功能说明 |
|------|------|----------|----------|
| FPGA | 高云GW2AR-LV18 | 逻辑资源 > 20k LUTs，支持PLL | DDS核心，时钟管理，数据处理 |
| MCU | STM32F411 | 32位ARM Cortex-M4处理器, 100MHz主频 | 系统控制，USB通信，用户接口 |
|时钟发生器| CDCI6214 | 4输出，支持外部参考 | 提供DAC时钟和系统时钟 |
| DAC | AD9767 | 14bit, 125MSPS, 双通道 | 波形生成核心，支持高精度输出 |
| 重构滤波器 | 5阶椭圆LPF | 截止频率35MHz，通带纹波<0.5dB | 滤除DAC采样频率成分，改善波形质量 |
| 差分放大器 | OPA695 | 电流反馈运算放大器、带宽>410MHz | 差分转单端 |
| 偏置DAC | DAC8562 | 16bit、输出噪声<90nV√Hz | 产生偏置电压 |
| 衰减网络 | 精密电阻、信号继电器 | 电阻精度<0.1% | 根据频率自动调整增益，实现幅度分段控制 |
| 输出功率级 | THS3095 |  | 驱动±10V@1MΩ负载 |

### 5.2 PCB要求
- 层数：≥4层（信号-地-电源-信号）
- 模拟板（FPGA+DAC）与控制板（STM32）分离，通过SPI接口连接
- DAC时钟线差分对（或单端包地），严格控制阻抗（50Ω）
- 模拟电源由LDO单独供电

## 7. 验收测试方法

| 参数 | 测试仪器 | 测试方法简述[^4] | 允差 |
|------|----------|--------------|------|
| 最大频率 | Siglent SDS824X | 正弦波30MHz，测量周期 | ±0.1% |
| 幅值精度 | Siglent SDS824X | 1kHz正弦，比较设定值与实测值 | 见3.2节 |
| THD | SDS824X (FFT模式) | 10MHz正弦，计算谐波功率 | ≤ -50dBc |
| 上升时间 | Siglent SDS824X | 方波5MHz，测量10%-90% | < 50ns |
| 频率精度 | Siglent SDS824X | 10MHz输出，24小时内漂移 | ±1ppm |
| 输出平坦度 | Siglent SDS824X|10MHz正弦，测量0.1MHz~10MHz范围内的幅值变化|< ±0.5dB|
| 抖动 (RMS) | SDS824X + 长时采集(>1000边沿) | 5MHz方波，测量边沿时间偏差标准差 | < 200 ps RMS |
| SCPI兼容 | PC + PyVISA | 发送`*IDN?`，校验返回格式 | 必须通过 |

[^4]: 测试时使用SMA->BNC短电缆，通过50Ω阻抗匹配器连接示波器，示波器开启全带宽。
