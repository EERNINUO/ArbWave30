## 1. 概述
- 软件组成：
    - MCU 固件（STM32F411）：SCPI 解析、USB 通信、FPGA 配置、UI 处理（按键/编码器）、非易失存储。
    - FPGA 逻辑（Verilog）：DDS 引擎、波形存储器、并口时序、SPI 从机接口。
    - 上位机测试脚本（Python + PyVISA，可选）。
- 开发环境：
  - MCU: Keil（MCU）
  - FPGA: GOWIN 云源
  - 测试脚本：Python 3.12

### 3.2 SCPI 解释器设计
状态机：采用 SCPI-1999 推荐的状态机（命令树遍历）。

支持的子系统（ V1.0 必须实现的）：

```text
*IDN?, *RST, *CLS
SOURce[1]:FUNCtion {SIN|SQU|TRI|SAW|STEp|DC}
SOURce[1]:FREQuency {<value>}
SOURce[1]:VOLTage {<value>} (Vpp)
SOURce[1]:VOLTage:OFFSet {<value>}
OUTPut[1]:STATe {ON|OFF}
OUTPut[1]:IMPedance {50|HIGH}
SYSTem:ERRor?
```

命令解析流程：接收字符 → 词法分析 → 查找命令树节点 → 执行回调 → 生成响应。

### 3.5 非易失存储

### 4.2 DDS 关键参数计算
系统时钟 Fclk = 125MHz
相位累加器位宽 N = 48
频率控制字 FTW = round( f_out * 2^N / Fclk )
<!-- 给出计算示例：f_out = 30MHz → FTW ≈ 30e6 * 2^48 / 125e6 = 0x28F5C28F5C28F5（可截断低8位？建议保留48位，精度满足1μHz）。 -->

### 4.3 波形生成
正弦波：查表（至少 4096 点，14bit 输出）。
方波：相位累加器 MSB 直接输出（再经幅度控制）。
三角波：相位累加器低 N-1 位经折叠变换。
锯齿波：直接取相位累加器高 14 位。
阶梯波：自定义（如每 16 个时钟改变一次输出值）。

### 4.4 DAC 时序
使用 PLL 产生的 125MHz 时钟作为 DAC 时钟。
数据更新：在每个时钟上升沿，将 dac_data 输出到引脚并保持满足建立时间。
如果需要双通道（未来），参考 AD9767 数据手册的并行模式。

### 4.5 SPI 从机接口
时钟速度 ≤ 10MHz。

状态机：空闲 → 接收命令码 → 接收参数 → 更新寄存器 → 发送 ACK（可选）。
