# ArbWave30 – 板间接口设计文档

| 文档编号 | ArbWave30-HDD-001 | 版本 | V1.0  |
| --------| ----------------- | ---- | ----- |
| 项目名称 | ArbWave30 | 日期 | 2026-04-24 |
| 起草人   | EERNINUO | 状态 | 草案 |

## 1. 概述
本文档定义了 ArbWave30 项目中**控制板**与**模拟板**之间所有物理和电气接口。控制板负责电源输入、初级转换、USB通信及 SCPI 解析；模拟板承载 FPGA、DAC 及模拟前端。

两板之间通过**两种连接器**互联：
- **主电源连接器 J_PWR**：传输 ±18V 功率电源（后续由 LDO 转为 ±15V 等）。
- **信号/辅助电源连接器 J_SIG**：传输数字控制信号（SPI、I2C、复位等）以及低压辅助电源（+3.3V_D）。

双方设计、布局、布线以及固件开发均须以本文档为唯一依据。

## 2. 连接器选型与机械要求

### 2.1 主电源连接器 (J_PWR)

| 项目 | 规格 |
|------|------|
| **类型** | XH2.54 间距，线对板，直插或弯插（根据结构决定） |
| **针数** | 8 针（预留冗余） |
| **额定电流** | 3A/针（实际需求 < 2A/针，充足余量） |
| **接触电阻** | ≤10mΩ |
| **耐压** | ≥250V |
| **工作温度** | -40℃ \~ +105℃ |
| **选型理由** | XH 连接器应用广泛、成本低廉，自带卡扣防止脱落。 |

### 2.2 信号/辅助电源连接器 (J_SIG)

| 项目 | 规格 |
|------|------|
| **类型** | DC2 牛角座，2.54mm 间距 |
| **针数** | 2×8P (16P)  |
| **额定电流** | 单针 1A（实际 < 200mA/针，降额充足） |
| **接触电阻** | ≤20mΩ |
| **工作温度** | -40℃ \~ +105℃ |
| **选型理由** | 带锁扣抗振动，镀金触点保证信号完整性；可同时传输低压电源。 |

### 2.3 机械装配要求

- **J_PWR 位置**：靠近板边，便于线束引出。
- **J_SIG 位置**：与 J_PWR 间隔至少 10mm，避免线束干涉。
- **排线长度**：J_SIG 使用的排线长度建议 ≤10cm，以保持 SPI 信号完整性。
- **防呆设计**：牛角座和 XH 连接器均有缺口定位，确保正确插入方向。

## 3. 引脚定义

### 3.1 主电源连接器 (J_PWR)

**引脚排列** 缺口朝上，左侧为1脚

| 引脚号 | 信号名 | 方向 | 额定电流 | 说明 |
|--------|--------|------|----------|------|
| 1 | 3.3V_D | 控制板 → 模拟板 | 1A | 数字电源，供 FPGA IO、 DAC 数字部分和模拟前端继电器使用 |
| 2 | GND_PWR | - | - | - | 功率地回路 |
| 3 | +17V_M | 控制板 → 模拟板 | 1A | 正电源轨，用于生成 +15V_A |
| 4 | GND_PWR | - | - | 功率地回路 |
| 5 | +17V_M | 控制板 → 模拟板 | 1A | 与引脚 3 并联，分摊电流 |
| 6 | -17V_M | 控制板 → 模拟板 | 0.6A | 负电源轨，用于生成 -15V_A |
| 5 | GND_PWR | - | - | 功率地回路 |
| 6 | -17V_M | 控制板 → 模拟板 | 0.6A | 与引脚 4 并联，分摊电流 |

### 3.2 信号/辅助电源连接器 (J_SIG)

**引脚排列**（2×8P，牛角座缺口朝左，1脚为左上，逆时针编号）

| 引脚 | 信号名 | 方向 | 电平 | 说明 |
|------|-------|------|------|-----|
| 1 | CS | 控制板 → 模拟板 | 3.3V | SPI 片选（低有效） |
| 2 | CLK | 控制板 → 模拟板 | 3.3V | SPI 时钟 |
| 3 | MISO | 模拟板 → 控制板 | 3.3V | SPI 从出主入 |
| 4 | MOSI | 控制板 → 模拟板 | 3.3V | SPI 主出从入 |
| 5 | NC | - | - | 预留 |
| 6 | FPGA_NRST | 控制板 → 模拟板 | 3.3V | 复位 FPGA（低有效）|
| 7 | GPIO1 | 双向 | 3.3V | 备用 GPIO |
| 8 | GPIO2 | 双向 | 3.3V | 备用 GPIO |
| 9 | GND | - | - | 地 |
| 10 | GND | - | - | 地 |
| 11 | GND | - | - | 地 |   
| 12 | GND | - | - | 地 |
| 13 | GND | - | - | 地 |
| 14 | GND | - | - | 地 |
| 15 | GND | - | - | 地 |
| 16 | GND | - | - | 地 |

> **注意**：+1.0V_D 电流较大，使用两个相邻引脚并联，并在模拟板入口处增加低 ESR 陶瓷电容（≥47μF）。

## 4. 电气特性规范

### 4.1 数字信号电平

| 参数 | 最小值 | 典型值 | 最大值 | 单位 |
|------|--------|--------|--------|------|
| VOH | 2.4 | 3.3 | - | V |
| VOL | - | 0 | 0.4 | V |
| VIH | 2.0 | - | 3.6 | V |
| VIL | -0.3 | - | 0.8 | V |
| 上升/下降时间 | - | - | 10 | ns |

### 4.2 电源轨纹波要求

| 电源轨 | 最大纹波 (峰峰值) | 测试条件 |
|--------|-------------------|----------|
| +3.3V_D | 50mV | 10Hz \~ 20MHz |
| +18V_M | 100mV | 10Hz \~ 20MHz |
| -18V_M | 100mV | 10Hz \~ 20MHz |

### 4.3 时序要求

- **SPI**：时钟频率 ≤10MHz，数据建立时间 ≥10ns，保持时间 ≥10ns。
- **FPGA_RST**：低电平脉冲宽度 ≥100ns。
- **上电顺序**：建议先有 +3.3V_D，后 +1.0V_D（FPGA 内核需要 IO 电源先上电）。

## 5. 电源去耦与滤波责任划分

| 位置 | 去耦要求 |
|------|----------|
| **控制板（输出侧）** | J_PWR 每个电源引脚附近放置 22μF 电容 + 0.1μF。J_SIG 的 +3.3V_D  同样放置 22μF + 0.1μF。 |
| **模拟板（输入侧）** | 连接器入口处每个电源引脚再放 10μF + 0.1μF。负载处设计详见[模拟板电源设计](./硬件设计文档(模拟板).md#模拟板电源设计) |

## 6. 接地与信号完整性策略

- **地平面**：两板均使用**完整地平面**，不分割数字地/模拟地。
- **接地引脚分配**：J_SIG 中每 2\~3 个信号引脚至少配一个 GND 引脚（已满足）。
- **电源回路**：±18V 的返回电流通过 J_PWR 的 GND 引脚回到控制板，不得与信号地混用同一引脚。
- **时钟信号**：125MHz DAC 时钟在模拟板上产生，不经过连接器。J_SIG 上的 SPI_CLK 频率较低（≤10MHz），无需特殊阻抗控制，但走线应尽量短。

## 7. 测试验证方法

| 测试项 | 方法 | 通过判据 |
|--------|------|----------|
| 连通性 | 万用表蜂鸣档 | 每个引脚接触电阻 < 1Ω |
| 电源纹波 | 示波器（地弹簧）测量模拟板入口 | 满足 4.2 节要求 |
| 信号质量 | 示波器测 SPI_CLK 上升沿（模拟板侧） | 过冲 < 0.5V，振铃 < 20% |
| 通信测试 | 控制板发送 SPI 命令，FPGA 回读 | 无误码 |
| 温升测试 | 满载运行 30 分钟，测连接器外壳温度 | < 50℃（环境 25℃） |
